# SAQL クエリ

**SAQL**（Salesforce Analytics Query Language）はすべてのレンズとダッシュボードを動かします。UI が自動で記述しますが、習得すると高度な計算が可能になります。

## クエリのパイプライン

SAQL はパイプラインのように読めます。各行はストリームを変換し、`q` に再代入します。

```sql
q = load "Opportunities";
q = filter q by 'StageName' == "Closed Won";
q = group q by 'Region';
q = foreach q generate 'Region' as 'Region', sum('Amount') as 'Revenue';
```

## グループ化と集計

`group` がグループを定義し、`foreach ... generate` が出力列を生成します。よく使う集計は `sum()`、`avg()`、`count()`、`unique()` です。

## 並べ替えと制限

`order q by 'Revenue' desc` と `limit q 10` を使うと、きれいな「トップ 10」の結果が得られます。
