# Données et jeux de données

Chaque jeu de données part de données brutes. Cette leçon explique comment les données arrivent et deviennent analysables.

## Connecter et synchroniser

Une **connexion** relie à une source : objets Salesforce, base de données ou fichier. Une **synchronisation** copie les champs nécessaires dans la plateforme selon une planification. Gardez les synchronisations légères : n'importez que les champs que vous utilisez réellement.

## Mettez-les en forme avec une recette

Une **recette** est l'outil visuel de préparation des données. Vous ajoutez des nœuds pour joindre, agréger, nettoyer et transformer les données, jusqu'à obtenir un jeu de données prêt à explorer.

## Dimensions, mesures et sécurité

Les jeux de données classent chaque champ comme **dimension** (catégories de regroupement), **mesure** (nombres à agréger) ou **date**. Un **prédicat de sécurité** est un filtre au niveau des lignes qui contrôle quelles lignes chaque utilisateur peut voir, par exemple `'OwnerId' == "$User.Id"`.
