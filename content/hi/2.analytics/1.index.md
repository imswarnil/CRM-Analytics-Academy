# SAQL क्वेरी

**SAQL** (Salesforce Analytics Query Language) हर लेंस और डैशबोर्ड को शक्ति देता है। इंटरफ़ेस इसे आपके लिए लिखता है, पर इसमें महारत उन्नत गणनाएँ खोल देती है।

## क्वेरी पाइपलाइन

SAQL एक पाइपलाइन की तरह पढ़ा जाता है। हर पंक्ति एक स्ट्रीम को रूपांतरित कर उसे फिर से `q` को सौंपती है:

```sql
q = load "Opportunities";
q = filter q by 'StageName' == "Closed Won";
q = group q by 'Region';
q = foreach q generate 'Region' as 'Region', sum('Amount') as 'Revenue';
```

## समूहन और एग्रीगेशन

`group` समूह तय करता है और `foreach ... generate` आउटपुट कॉलम बनाता है। सामान्य एग्रीगेशन हैं `sum()`, `avg()`, `count()` और `unique()`।

## क्रमबद्ध और सीमित करें

`order q by 'Revenue' desc` और `limit q 10` का उपयोग करके साफ़-सुथरा "टॉप 10" परिणाम लौटाएँ।

## आगे क्या

इसके बाद आप इन क्वेरीज़ को इंटरैक्टिव डैशबोर्ड और पूर्वानुमानों में बदलेंगे।
