# Dados e conjuntos de dados

Todo conjunto de dados começa com dados brutos. Esta lição mostra como os dados entram e se tornam algo que você pode analisar.

## Conectar e sincronizar

Uma **conexão** liga a uma origem: objetos do Salesforce, um banco de dados ou um arquivo. Uma **sincronização** copia os campos necessários para a plataforma conforme uma programação. Mantenha as sincronizações enxutas: traga só os campos que você realmente usa.

## Modele com uma receita

Uma **receita** é a ferramenta visual de preparação de dados. Você adiciona nós para unir, agregar, limpar e transformar os dados, até produzir um conjunto de dados pronto para explorar.

## Dimensões, medidas e segurança

Os conjuntos de dados classificam cada campo como **dimensão** (categorias para agrupar), **medida** (números para agregar) ou **data**. Um **predicado de segurança** é um filtro em nível de linha que controla quais linhas cada usuário pode ver, por exemplo `'OwnerId' == "$User.Id"`.
