# Consultas SAQL

O **SAQL** (Salesforce Analytics Query Language) alimenta cada lente e painel. A interface o escreve para você, mas dominá-lo libera cálculos avançados.

## O pipeline de consulta

O SAQL é lido como um pipeline. Cada linha transforma um fluxo e o reatribui a `q`:

```sql
q = load "Opportunities";
q = filter q by 'StageName' == "Closed Won";
q = group q by 'Region';
q = foreach q generate 'Region' as 'Region', sum('Amount') as 'Revenue';
```

## Agrupar e agregar

`group` define os grupos e `foreach ... generate` produz as colunas de saída. As agregações comuns são `sum()`, `avg()`, `count()` e `unique()`.

## Ordenar e limitar

Use `order q by 'Revenue' desc` e `limit q 10` para retornar um resultado limpo de "10 primeiros".
