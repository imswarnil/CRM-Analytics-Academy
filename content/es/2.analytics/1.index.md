# Consultas SAQL

**SAQL** (Salesforce Analytics Query Language) impulsa cada lente y panel. La interfaz lo escribe por ti, pero dominarlo te permite hacer cálculos avanzados.

## La canalización de consultas

SAQL se lee como una canalización. Cada línea transforma un flujo y lo reasigna a `q`:

```sql
q = load "Opportunities";
q = filter q by 'StageName' == "Closed Won";
q = group q by 'Region';
q = foreach q generate 'Region' as 'Region', sum('Amount') as 'Revenue';
```

## Agrupar y agregar

`group` define los grupos y `foreach ... generate` produce las columnas de salida. Las agregaciones comunes son `sum()`, `avg()`, `count()` y `unique()`.

## Ordenar y limitar

Usa `order q by 'Revenue' desc` y `limit q 10` para devolver un resultado limpio de los "10 primeros".

## Qué sigue

A continuación convertirás estas consultas en paneles interactivos y predicciones.
