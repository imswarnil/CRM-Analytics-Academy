# Datos y conjuntos de datos

Todo conjunto de datos parte de datos sin procesar. Esta lección explica cómo entran los datos y se convierten en algo que puedes analizar.

## Conectar y sincronizar

Una **conexión** enlaza con un origen: objetos de Salesforce, una base de datos o un archivo. Una **sincronización** copia los campos que necesitas en la plataforma según una programación. Mantén las sincronizaciones ligeras: trae solo los campos que realmente usas.

## Dale forma con una receta

Una **receta** es la herramienta visual de preparación de datos. Añades nodos para unir, agregar, limpiar y transformar los datos, hasta producir un conjunto de datos listo para explorar.

## Dimensiones, medidas y seguridad

Los conjuntos de datos clasifican cada campo como **dimensión** (categorías por las que agrupas), **medida** (números que agregas) o **fecha**. Un **predicado de seguridad** es un filtro a nivel de fila que controla qué filas ve cada usuario, por ejemplo `'OwnerId' == "$User.Id"`.
