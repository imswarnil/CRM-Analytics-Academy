# SAQL 查询

**SAQL**（Salesforce Analytics Query Language）驱动着每个透镜和仪表板。界面会替你编写它，但掌握它能解锁高级计算。

## 查询管道

SAQL 读起来像一条管道。每一行都会转换一个流并重新赋值给 `q`：

```sql
q = load "Opportunities";
q = filter q by 'StageName' == "Closed Won";
q = group q by 'Region';
q = foreach q generate 'Region' as 'Region', sum('Amount') as 'Revenue';
```

## 分组与聚合

`group` 定义分组，`foreach ... generate` 生成输出列。常用的聚合有 `sum()`、`avg()`、`count()` 和 `unique()`。

## 排序与限制

使用 `order q by 'Revenue' desc` 和 `limit q 10` 即可返回简洁的"前 10 名"结果。
