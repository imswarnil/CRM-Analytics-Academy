# Get Access to CRM Analytics

Before you can build anything, an admin needs to enable the platform and assign a license.

## Enable the platform

From **Setup**, search for **Analytics** and turn on CRM Analytics for the org. This provisions the underlying analytics infrastructure — it can take a few minutes on a fresh org.

## Assign a permission set

CRM Analytics access comes from a **permission set license** plus a **permission set** (commonly `CRM Analytics Plus Admin` or `CRM Analytics Plus User`). Assign one to your user in **Setup → Users**.

## Open CRM Analytics

Use the **App Launcher** (the grid icon in the top-left) and search for **Analytics Studio**. This is the home screen for everything you'll build in this course.
