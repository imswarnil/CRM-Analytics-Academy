# Create Your First App

An **app** in CRM Analytics is a container — think of it as a shared folder for related dashboards, datasets, and recipes.

## Create the app

In Analytics Studio, select **Create → App**, then choose **Blank App** to start empty. Give it a clear name; you'll reference it constantly while building.

## Set sharing

Apps have their own sharing model, separate from Salesforce object permissions. Open the app, go to **Share**, and add users or groups as **Viewer**, **Editor**, or **Manager**.

## Add assets to the app

Anything you create — a dataset from a recipe, a dashboard, a lens — gets saved into an app. Keep related work together so sharing stays simple.

## What you'll learn next

With an app in place, the next lesson tours the CRM Analytics Studio interface so you know where everything lives.
