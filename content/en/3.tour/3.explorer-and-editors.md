# Explorer & Dashboard Editor

## Explorer

**Explorer** opens a dataset as a **lens** — an ad-hoc, chart-driven query surface. Drag fields into groupings, switch chart types, and apply filters without writing SAQL by hand. Every chart in Explorer can be clipped straight onto a dashboard.

## Dashboard Editor

The **Dashboard Editor** has three main areas:

- **Canvas** — the free-form or grid layout where widgets live.
- **Widget panel** — add charts, lenses, number widgets, list widgets, images, and containers (tabs, flex layouts).
- **Bindings/Interactions panel** — wire widgets together so a click in one filters another (covered in depth in the Bindings section).

## Preview vs. Edit mode

Toggle between **Edit** (building layout, adding widgets) and **Preview** (testing exactly what a viewer sees, including facets and actions) as you work.
