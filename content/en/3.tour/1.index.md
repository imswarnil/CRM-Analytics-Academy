# Analytics Studio Home

Analytics Studio is the single home screen for building and managing everything in CRM Analytics.

## Top navigation bar

- **Home** — recent and favorite assets.
- **Browse** — every app, dataset, dashboard, and recipe you have access to, with search and filters.
- **Create** — the entry point for new apps, dataflows, recipes, datasets, dashboards, and Einstein Discovery stories.

## The asset library

**Browse** lists assets in a table you can filter by type (App, Dashboard, Dataset, Lens, Recipe...), owner, and last-modified date — the fastest way to find something once an org has dozens of dashboards.

## The Create menu

Every asset type starts from the same **Create** button in the top-right. This course revisits it repeatedly: recipes in the data lessons, dashboards in the navigating lessons, and stories when you reach Einstein Discovery.
